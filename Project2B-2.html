Database

The databaes will be a somewhat simple model. It has quite a few tables, but each of them is essential. I currently plan for there to be an Account table to store account information, Character table to store chracter information for the various accounts, and a Gear Set table to store the saved gear set data for each character on the account. In addition, an Item List table for the blank shells of the in-game items, Classes table to store the various character classes, Abilities table to store character abilities in a way accessible for computation, and a Character Buffs table meant to store the various permanent buffs a character might receive will also be needed.

The database will be accessed via SQL queries performed either by JavaScript or by the various php forms the site intends to utilize, and will be updated in the same manner.
