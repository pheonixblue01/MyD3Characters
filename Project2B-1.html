Website Plan



  • Conceptual Design

    The website's conceptual design illustrates each individual intended section of the website as it's currently planned. The image displayed will provide the model the site is built from.

  • Website Plan

    The website plan will provide details on which multimedia files and forms will be used for each HTML file as is currently intended.

  • Website Structure

    The structure section provides information on how each form is placed and shows the connections between different website sections as is currently planned.

  • Multimedia

    This section will provide a list for each individual page to show which media types will be used. The components include whether or not it has any multimedia functionality such as images, sound, music files, or the like.

  • Database

    The database section provides a diagram of how the database structures the information required to support the various features of the site, as well as how it is sorted. The provided model will illustrate the design for each user's profile section.
